# 标准依赖

StarCraft II三部曲中使用的每个数据资产都通过依赖模块进行分发。这些标准依赖建立在游戏的基础 Core.SC2Mod 的基础上，并提供与特定版本的 StarCraft 相关的资产。本文详细介绍了每个标准依赖，包括其目的和内容的信息，无论是资产还是其他依赖。
